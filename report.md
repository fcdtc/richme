  ✅ 可用数据源

  1. 新浪财经 K线接口（推荐）

  接口地址：
  https://quotes.sina.cn/cn/api/json_v2.php/CN_MarketDataService.getKLineData

  参数：

  ┌─────────┬────────────────────────┬────────────────────────────────────────┐
  │  参数   │          说明          │                  示例                  │
  ├─────────┼────────────────────────┼────────────────────────────────────────┤
  │ symbol  │ 股票代码（带市场前缀） │ sh512400（上交所）/ sz159915（深交所） │
  ├─────────┼────────────────────────┼────────────────────────────────────────┤
  │ scale   │ K线周期（分钟数）      │ 240=日K, 60=60分钟, 30=30分钟          │
  ├─────────┼────────────────────────┼────────────────────────────────────────┤
  │ datalen │ 返回数据条数           │ 30                                     │
  └─────────┴────────────────────────┴────────────────────────────────────────┘

  测试结果：
  [{"day":"2026-01-16","open":"2.206","high":"2.249","low":"2.163","close":"2.175","volume":"1456648009",...}]

  优点：
  - 无需认证
  - 返回JSON格式，易于解析
  - 包含MA5/MA10/MA30均线数据
  - 稳定性高

  2. 腾讯财经 K线接口

  接口地址：
  https://web.ifzq.gtimg.cn/appstock/app/fqkline/get

  参数：

  ┌───────┬─────────────────────────────────────────────────┬───────────────────────┐
  │ 参数  │                      说明                       │         示例          │
  ├───────┼─────────────────────────────────────────────────┼───────────────────────┤
  │ param │ 格式：市场代码,周期,开始日,结束日,数量,复权类型 │ sh512400,day,,,30,qfq │
  ├───────┼─────────────────────────────────────────────────┼───────────────────────┤
  │ r     │ 随机数（防缓存）                                │ 时间戳                │
  └───────┴─────────────────────────────────────────────────┴───────────────────────┘

  测试结果：
  {"code":0,"data":{"sh512400":{"qfqday":[["2026-01-16","2.206","2.175","2.249","2.163","14566480.000"],...]}}}

  数据格式： [日期, 开盘, 收盘, 最高, 最低, 成交量(万手)]

  优点：
  - 支持前复权(qfq)、后复权(hfq)
  - 支持日K/周K/月K
  - 响应速度快
